Skills

Technical Skills

  • Dribbling: ความสามารถในการเลี้ยงบอลผ่านคู่แข่งด้วยความรวดเร็วและแม่นยำ
  • Finishing: ทักษะการทำประตูที่เฉียบขาดในทุกสถานการณ์
  • Playmaking: การสร้างโอกาสให้เพื่อนร่วมทีมและการอ่านเกมที่ยอดเยี่ยม
  • Free-kick: ความแม่นยำในการยิงฟรีคิกจนเป็นที่ยอมรับทั่วโลก

Soft Skills

  • Leadership: ความเป็นผู้นำที่ช่วยยกระดับทีมทั้งในระดับสโมสรและทีมชาติ
  • Teamwork: การประสานงานกับเพื่อนร่วมทีมเพื่อสร้างความสำเร็จ
  • Discipline: ความมุ่งมั่นและความทุ่มเทในการฝึกซ้อมและการเล่น
  • Emotional Strength: การรับมือกับความกดดันในสถานการณ์สำคัญ
